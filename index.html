Short Filter プライバシーポリシー

最終更新日: 2026-03-21

導入ページをひらく

Short Filter（以下「本アプリ」）は、保護者が Android 端末上で YouTube の短尺動画導線を制限するための設定アプリです。

1. 取得する情報

本アプリは、以下の目的で端末上の情報を利用します。

  • Accessibility Service を通じた画面上の UI 情報
  • アプリ内で設定したフィルタ設定
  • 親 PIN の検証に必要なローカル保存データ

2. 情報の利用目的

  • YouTube の Shorts 導線や短い動画カードを検出するため
  • 親が設定した制限ルールを端末内で適用するため
  • 設定変更時に親 PIN を確認するため
  • アプリの基本的な動作状態を端末内で表示するため

3. 外部送信について

本アプリは、次の情報を外部サーバーへ送信しません。

  • 動画タイトル
  • 動画本文
  • 検索語
  • 閲覧履歴
  • 親 PIN の平文

本アプリは、広告 SDK や外部分析 SDK を含みません。

4. データ保存場所

本アプリの設定データは、原則として端末内にのみ保存されます。

5. 保存期間と削除

  • 設定データは、アプリを使用している間、端末内に保存されます
  • アプリをアンインストールした場合や端末上でアプリデータを削除した場合、ローカル保存された設定は削除されます
  • 開発者は、端末外に送信していないデータをサーバー上で保持しません

6. アカウントについて

本アプリは、アカウント作成やログインを必須としません。

7. 子ども向けサービスではありません

本アプリは、保護者が端末設定を行うためのツールです。子ども向けの娯楽アプリとして提供するものではありません。

8. お問い合わせ

開発者名、公開用メールアドレス、連絡先 URL（任意）をここに記載してください。

9. 公開場所について

Play Console に登録する際には、このポリシーを公開された HTTPS URL で参照できる状態にしてください。ローカルファイルや PDF のままでは登録できません。

10. 改定

本ポリシーは、必要に応じて改定されることがあります。
